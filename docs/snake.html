ヒント 1 — ゲームの仕組みと SnakeHID クラス
Web ページがゲームを進行し、UIAPduino は 次の方向 を返すだけです。

通信の流れ:
CMD_START → UIAPduino が初期化して最初の方向を送信
CMD_TICK → 1マス移動するたびに呼ばれる。次の方向を送信
CMD_DIR → UIAPduino が Web に返す移動方向

SnakeHID クラスのメソッド:
snake.sendDir(dx, dy) — 次の方向を送信（必ず呼ぶ）

方向の表現:
dx=+1, dy= 0 → 右   dx=-1, dy= 0 → 左
dx= 0, dy=+1 → 下   dx= 0, dy=-1 → 上
ヒント 2 — ゲーム状態の管理
UIAPduino は自分で盤面の状態を管理します。
CMD_TICK で受け取る新しい頭の座標を記録し続けます:
bool    board[16][16]; // true = 占有済み
uint8_t headX, headY;  // 現在の頭の座標

// 初期化（CMD_START 受信時）
void initBoard(uint8_t sx, uint8_t sy) {
    for (uint8_t y = 0; y < 16; y++)
        for (uint8_t x = 0; x < 16; x++)
            board[y][x] = false;
    headX = sx; headY = sy;
    board[sy][sx] = true;
}

// 更新（CMD_TICK 受信時）
headX = buf[1]; headY = buf[2];
board[headY][headX] = true;
次の移動先が board[ny][nx] なら衝突するので避ける必要があります。
ヒント 3 — シンプルなアルゴリズム（ランダム・壁沿い）
まずは「衝突しない方向をランダムに選ぶ」実装から始めてみましょう:
void computeNextDir(int8_t& outDx, int8_t& outDy) {
    const int8_t DDX[] = { 1, 0, -1,  0 };
    const int8_t DDY[] = { 0, 1,  0, -1 };

    // ランダムな順序で4方向を試す
    uint8_t order[] = { 0, 1, 2, 3 };
    // （簡易シャッフル）
    for (uint8_t i = 3; i > 0; i--) {
        uint8_t j = myRand(i + 1);
        uint8_t tmp = order[i]; order[i] = order[j]; order[j] = tmp;
    }

    for (uint8_t i = 0; i < 4; i++) {
        int8_t nx = (int8_t)headX + DDX[order[i]];
        int8_t ny = (int8_t)headY + DDY[order[i]];
        if (nx < 0 || nx >= 16 || ny < 0 || ny >= 16) continue;
        if (board[ny][nx]) continue;
        outDx = DDX[order[i]];
        outDy = DDY[order[i]];
        return;
    }
    // 逃げ場なし（ゲームオーバーになる）
    outDx = 1; outDy = 0;
}
これだけでも動きますが、すぐに行き詰まります。全マスを埋めるにはヒント 4 を参照。
ヒント 4 — ハミルトンパス（全マス確実に埋めるアルゴリズム）
16×16 グリッドを蛇行するルートを辿ることで、どこからスタートしても全マスを埋められます:

偶数行: 左 → 右 (x: 0→15)
奇数行: 右 → 左 (x: 15→0)

現在位置を経路番号（0〜255）に変換し、次の番号の座標を計算します:
void computeNextDir(int8_t& outDx, int8_t& outDy) {
    // 現在位置を経路番号に変換
    uint16_t pos;
    if (headY % 2 == 0) {
        pos = (uint16_t)headY * 16 + headX;        // 偶数行: 左→右
    } else {
        pos = (uint16_t)headY * 16 + (15 - headX); // 奇数行: 右→左
    }

    // 次の番号を座標に変換
    uint16_t nextPos = (pos + 1) % 256;
    uint8_t  ny = (uint8_t)(nextPos / 16);
    uint8_t  nx = (ny % 2 == 0)
                  ? (uint8_t)(nextPos % 16)         // 偶数行
                  : (uint8_t)(15 - nextPos % 16);   // 奇数行

    outDx = (int8_t)nx - (int8_t)headX;
    outDy = (int8_t)ny - (int8_t)headY;
}
メモリ使用量: ほぼゼロ（配列不要・計算のみ）。
